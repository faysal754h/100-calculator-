How to Use the Event Planning Calculator

  1. Enter the number of guests for your event.
  2. Input costs (e.g., food, drinks, venue, decorations, other) and quantities (e.g., food portions, drink servings) per guest.
  3. Click "Calculate" to see the total cost, per-person cost, and estimated quantities.
  4. After 3 calculations, you will be redirected to an ad. Return to continue using the calculator.
