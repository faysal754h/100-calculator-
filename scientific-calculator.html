How to Use the Scientific Calculator

  1. Select the operation (e.g., addition, sine, logarithm).
  2. Enter the required input(s). Use one input for square root, trig, or logs; two inputs for arithmetic or custom base log.
  3. Click the "Calculate" button to see your result.
  4. After 3 calculations, you will be redirected to an ad. Return to continue using the calculator.
